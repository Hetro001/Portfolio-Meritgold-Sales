Project 01

Responsive Website Clone

Project Overview

A professional recreation of an existing website built entirely from scratch using HTML and CSS. The goal was to replicate the design accurately while ensuring the result was fully responsive across all screen sizes and devices.

Client Goal

The client needed a clean, responsive website that matched a reference design closely — demonstrating that professional web results can be achieved without expensive platforms or tools.

My Approach

I carefully analyzed the reference design, broke it into sections, and rebuilt each part using semantic HTML and modern CSS — including Flexbox and Grid for layout, and media queries for full responsiveness.

Tools Used

HTML5 CSS3 Flexbox CSS Grid Responsive Design

Outcome

Delivered a pixel-accurate, fully responsive website clone that performs consistently across desktop, tablet, and mobile — with clean, maintainable code ready for further development.
